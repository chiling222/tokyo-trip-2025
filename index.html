重要
護照效期確認（6個月以上）

Visit Japan Web 預先填寫
日本入境申報可提前線上填寫，節省機場排隊時間。建議在台灣出發前完成
→ Visit Japan Web 官網
日本禁帶入境物品確認
禁帶：生鮮肉類、蔬果、活體動植物。藥品限個人用量，處方藥需備文件
日幣換匯
建議出發前在台灣換好日幣（台灣銀行匯率通常較優），或抵達成田後在機場 7-11 ATM 提領。建議每人備用 ¥30,000–50,000 現金
網路 SIM 卡或 WiFi 分享器
日本吃到飽 SIM 或 eSIM，5人各自一張最方便。可在台灣機場領取或提前購買
